# Projeto

KEBA8 é um projeto que visa dar um feedback visual para donos de plantinhas que querem saber quando regar sua planta. 

# Componentes

 - Arduino Mega  2560
 - Jumpers
 - Display LCD 20x4 com LCI
 - Sensor de umidade do solo
 - Protoboard mini
 - Papel alumínio
